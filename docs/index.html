About the MARKUP Theme¶

The MARKUP theme is a theme for use with Sphinx projects. The MARKUP theme is open source and is uses the Apache License 2.0 license.

This site shows off the capabilities of the MARKUP theme, including:

  • Multiple sites that act as a single collection of documentation
  • Custom left-side and top-level navigation
  • Search
  • Hosted on GitHub pages
  • Formatting guides for both Markdown and reStructuredText authoring
  • Tutorials for extending the MARKUP theme
  • Documentation of various components within the MARKUP theme
